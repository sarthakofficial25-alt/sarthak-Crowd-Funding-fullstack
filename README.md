# 🚀 Crowd Funding Smart Contract (Soroban)

## 📌 Project Description
This project is a basic Crowd Funding smart contract built using Soroban on the Stellar blockchain. It allows users to create funding campaigns and receive contributions from anyone in a fully decentralized and permissionless manner.

---

## ⚙️ What it does
- Users can create a crowdfunding campaign by setting a funding goal.
- Anyone can contribute funds to any campaign.
- The contract keeps track of total funds raised for each campaign.
- Users can query campaign goals and collected funds at any time.

---

## ✨ Features
- 🔓 Permissionless campaign creation (anyone can create)
- 💸 Open contributions (anyone can fund any campaign)
- 📊 Transparent fund tracking
- ⚡ Built on Stellar Soroban (fast & low-cost)
- 🧩 Simple and extensible architecture
